Lambda Calculus features

Functions are first-class

functions can be used as inputs to other functions, and functions can return functions


                let add2 = (x) => x + 2;
                (f) => f();
                (x) => return f;

Anonymous Functions

Functions are always anonymous


            let add2 = (x) => x + 2;

Unary Functions

Functions in lambda calculus only accept a single input.
If you need more than one parameter, the function will take one input and return a new function that takes the next. 


                (x, y) => x + y;
                (x) => (y) => x + y;
