Curried Function

A function that will return a new function
until it receives all its arguments

            
                    let curry = require('lodash/curry');
                    
                    let match = curry((what, str) => str.match(what))
                    
                    let hasSpaces = match(/\s+/g);
                    hasSpaces("Hello World"));

                    let hasColon=match(/;+/g)
                    hasColon("Hello World;");
