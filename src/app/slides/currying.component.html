Curried Function

A function that will return a new function
until it receives all its arguments

Currying | Benefits

Make new functions just by giving our base functions

                        
                                let curry = require('lodash/curry');
                                
                                let match = curry((what, str) => str.match(what))
                                
                                let hasSpaces = match(/\s+/g);
                                hasSpaces("Hello World"));
            
                                let hasColon=match(/;+/g)
                                hasColon("Hello World;");

Currying | Benefits

Transform any function that works on single elements into a function that works on arrays 

                                
                                        let curry = require('lodash/curry');
                                        let add = curry ((x, y) => x + y)
                                        
                                        let increment = add(1);
                                        let myArray = [10, 20, 30, 40, 50]
                                        
                                        let newArray = myArray.map(increment);
                                        
                                        console.log(newArray)
                                        //[ 11, 21, 31, 41, 51 ]
