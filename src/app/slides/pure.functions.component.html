Pure Function

A function that, given the same input, will always return the same output and does not have any observable side effect.


                    let abcd = ['a', 'b', 'c', 'd', 'e']
                    console.log(abcd.slice(2,4))
                    //[ 'c', 'd' ]
                    console.log(abcd)
                    //[ 'a', 'b', 'c', 'd', 'e' ]
                    console.log(abcd.splice(2,2))
                    //[ 'c', 'd' ]
                    console.log(abcd)
                    //[ 'a', 'b', 'e' ]

Properties of Pure fuctions

1. Immutability
    In Pure FP, X = X + 1 is not possible
2. No side effects
3. Referrential Transparency
    let plusOne = (x) => x+1;
    plusOne(5) can be replaced by "6"
4. Parallel Code
